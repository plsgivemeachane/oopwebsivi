# Sivi

This project is a copy from scratch of nginx server.

## Setup Instructions

I. Clone the repository:
```bash
git clone https://github.com/plsgivemeachane/oopwebsivi.git
```

II. Install dependencies:
```bash
npm install
```

## Usage

Run the server:
```bash
ts-node index.ts
```
